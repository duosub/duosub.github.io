üìñ User Guide: Getting Started

DuoSub is designed as a Single Purpose tool: to convert and translate subtitle files locally without data collection.

1

Upload File

Click the Folder Icon (üìÇ) area. Select your subtitle file. We support UTF-8 encoded .srt, .vtt, .ass, .sbv, .sub, and .txt.

2

3

Click Translate. Watch the live preview. Once finished (100%), click Download to get your new Dual Subtitle file.

‚ö†Ô∏è How to Fix "Error: ALL_KEYS_DEAD"

Important

This error means the community shared keys have reached their daily limit with Google. To fix this permanently, create your own free Personal API Key.

Step-by-Step Instructions

  1. Go to script.google.com and sign in.
  2. Click + New Project.
  3. Delete the existing code and paste the code block below:
function doPost(e) {
  var p = JSON.parse(e.postData.contents);
  var translated = LanguageApp.translate(p.q, "", p.target);
  return ContentService.createTextOutput(JSON.stringify({
    status: "success", 
    translatedText: translated
  })).setMimeType(ContentService.MimeType.JSON);
}

    • Click Deploy → Authorize access → Copy the Web App URL.
  2. Open the extension, click "‚öôÔ∏è Use Personal API Key", and paste your URL.

üß† Language Learning Strategies

Maximize your learning using Dual Subtitles with these proven methods:

1. The Shadowing Technique

Method: Listen to the audio and read the Target Language subtitle simultaneously. Repeat what you hear immediately.

Benefit: Improves pronunciation, prosody, and speaking speed.

2. Dual Coding & Context

Method: Use the Original Language subtitle only when you don't understand a word. Treat it as an instant dictionary.

Benefit: Creates strong neural links between the visual context (video) and the new vocabulary.

3. Passive Immersion

Method: Watch content comfortably with Dual Subs. Your brain subconsciously absorbs sentence structures.

Benefit: Reduces "learning fatigue" allowing for longer study sessions.
