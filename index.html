About the Extension

Duosub SRT is a cross-platform browser extension designed to enhance language learning and accessibility. It allows users to overlay custom SRT subtitle files onto any video player. By providing simultaneous display of two languages, it bridges the gap between native content and learner comprehension.

Technical Implementation

  • Engine: Built on Manifest V3 for maximum security and performance.
  • Rendering: Uses a MutationObserver to detect video player elements and inject a custom subtitle container without interfering with the site's original code.
  • Performance: Lightweight Vanilla JS implementation ensures zero impact on CPU/RAM while watching high-definition video.
  • Cross-Platform: Responsive design ensures the settings UI is accessible on desktop, mobile, and tablets.
